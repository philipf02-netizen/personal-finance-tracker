# Personal Finance Tracker

A full-featured personal finance web app built with Next.js 15, Tailwind CSS, and shadcn/ui.

## Features

### Budget Tracker
- Track income and expenses by category
- Expense pie chart & income vs expenses bar chart
- Category spending progress bars
- Persistent data via localStorage

### Debt Payoff Planner
- Add multiple debts (balance, APR, minimum payment)
- Snowball vs. Avalanche method comparison
- Extra payment calculator
- Payoff timeline chart
- Side-by-side interest cost comparison

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack
- Next.js 15 (App Router) + React 19
- TypeScript + Tailwind CSS
- shadcn/ui + Recharts + Lucide React
